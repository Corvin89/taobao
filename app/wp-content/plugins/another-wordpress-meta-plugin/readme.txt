== Description ==

**This plugin is no longer supported or maintained.**

Add META tags like keywords and descriptions to your posts and pages.

If you'd like to optimize your blog for search engines,
[All in One SEO Pack](http://wordpress.org/extend/plugins/all-in-one-seo-pack/)
might be helpful too ...

After activation you have several additional fields when editing/creating pages and posts. Just insert
your keywords, descriptions etc. there.

== Installation ==

1. Upload `another_wordpress_meta_plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!
