== Installation ==

Download, Upgrading, Installation:

Upgrade

1. First deactivate WP Super Secure and Fast htaccess
1. Remove the `wp-super-htaccess` directory

**Install**

1. Unzip the `wp-super-htaccess.zip` file. 
1. Upload the the `wp-super-htaccess` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the WP Super Secure and Fast htaccess plugin and a subpage for Super htaccess will appear in your Options menu.
