== Description ==

Auto add keywords and description in the head meta of home,single,page,category and tag. Help seo for head meta tag.
Languages:English, Chinese simplified(zh_CN).
The illustration is available here:<a href="http://www.gengtang.net/wordpress-plugin/another-wordpress-seo-plugin.html" target="_blank">another-wordpress-seo-plugin</a>, or in AWSP-option after installation. If you translate it, please notice me.

== Installation ==

1. Upload the folder another-wordpress-seo-plugin to the `/wp-content/plugins/
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Manage > Option > AWSP-option,the illustration is available there.
